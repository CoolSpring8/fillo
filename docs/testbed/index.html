Fillo Form Testbed

Hosted fixtures that mimic real-world applicant tracking system forms. Use this testbed with the Fillo extension to validate field detection, autofill coverage, and instrumentation.

Scenarios

  • Simple baseline form
  • Messy form (missing labels)
  • Wizard (multi-step)
  • Array inputs (repeaters)
  • Address combinations
  • Segmented phone fields
  • File upload + cover letter
  • Shadow DOM variants
  • Same-origin iframe
  • Cross-origin iframe
  • Chinese labels

Usage

  1. Publish this folder via GitHub Pages (e.g., docs/ on main or gh-pages).
  2. Grant your extension host access to the published origin.
  3. Load the Fillo extension and open one of the fixtures above.
  4. Use the Test Harness panel (top-right) to randomize DOM structure, load sample resume data, and diff expected vs. actual autofill values.
